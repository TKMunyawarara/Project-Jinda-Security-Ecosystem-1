# Project Jinda: SOC Security Monitor

Project Jinda is a lightweight Security Operations Center (SOC) application engineered to detect, analyze, and mitigate threats across three tactical pillars.

## Core Pillars
* **Email Security Module:** Analyzes mail flow, detects phishing vectors, identifies malicious attachments, and validates domain authentication (SPF/DKIM/DMARC).
* **Ports & Protocols Monitor:** Tracks network surface exposure, monitors unauthorized open ports, and flags anomalous traffic behaviors or protocol manipulation.
* **IPS (Intrusion Prevention System) Rules Engine:** Implements deep packet inspection and custom signature-based matching to block active exploits and network-layer attacks.
